# reserpital 



## サイト概要
### サイトテーマ
病院の診察の予約サイトです。

### テーマを選んだ理由
小さい子供を連れて病院に行くと子供が長時間待っていられないので、
　診察予約ができればいいのにと常々思っていたのでこのテーマにしました。

### ターゲットユーザ
- 病院での待ち時間を短縮したい人
- 大体の待ち時間を知りたい人

### 主な利用シーン
- 病院に行く時
- 大体の待ち時間を知りたい時
